class workspace


A class that represents the workspace. This includes maintaining information about the notes and undo/redo information.

createNote

Object createNote(<dictionary> note)
    Create a new note. Any of the parameter values may be left blank and default values will be used.
    Parameters:
      note - a dictionary with any of the following keys: note['id'] = the name of the note
      note['xPos'] = the x position of the note
      note['yPos'] = the y position of the note
      note['height'] = the height of the note
      note['width'] = the width of the note
      note['bgcolor'] = the background color of the note (hex)
      note['zIndex'] = the stacking position of the note
      note['text'] = the text value of the note
